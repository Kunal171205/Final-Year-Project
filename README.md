# Industry Explorer with GEO AI

A Flask-based web application for job portal and B2B marketplace with geolocation features.

## Features
- User authentication (Workers and Businesses)
- Job portal for workers
- Job posting for businesses
- B2B marketplace
- User profiles and company profiles

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser to `http://localhost:5000`

## Deployment

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed deployment instructions.

**Note:** This Flask application cannot be hosted on GitHub Pages. Use platforms like Render, Railway, or Heroku for deployment.
